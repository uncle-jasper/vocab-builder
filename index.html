One-time setup in Anki. Creates a Note Type with two card templates: Word → Definition and Definition → Word. Each note you import automatically generates both cards.
Step 1 — Create the Note Type
In Anki: Tools → Manage Note Types → Add → Add: Basic → OK
Name it Vocab Builder → OK
Step 2 — Set up fields
Select Vocab Builder → Fields
Delete any existing fields, then add three fields in this order:
Word  ·  Definition  ·  Example
Step 3 — Card Template 1: Word → Definition
Select Vocab Builder → Cards → rename to Word → Definition
Paste these into the front and back fields:
Front Template
{{Word}} {{#Audio}}{{Audio}}{{/Audio}}
Back Template
{{FrontSide}}
{{Definition}}
{{Example}}
Step 4 — Card Template 2: Definition → Word
Still in the Cards editor → Options → Add Card Type → name it Definition → Word
Front Template
{{Definition}}
Back Template
{{FrontSide}}
{{Word}} {{#Audio}}{{Audio}}{{/Audio}}
{{Example}}
Step 5 — Audio with AwesomeTTS
After importing new cards:
1. In the card browser, select your new cards
2. AwesomeTTS → Add Audio to Selected
3. Source field: Word → Destination field: Audio
4. Choose a TTS service (Google TTS works well) → Generate

This fills the Audio field with [sound:word.mp3] tags, which both card templates display automatically.
Step 6 — Importing cards
File → Import → select the exported .txt file
Set Note Type to Vocab Builder
Separator: Tab
Map fields: Field 1 → Word, Field 2 → Definition, Field 3 → Example
Check Update existing notes where first field matches — this prevents duplicates at the Anki level too.

Run AwesomeTTS on the newly imported cards to generate audio.
