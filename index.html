One-time setup in Anki. Creates a Note Type with two card templates: Word → Definition and Definition → Word. Each note you import automatically generates both cards.
Step 1 — Create the Note Type
In Anki: Tools → Manage Note Types → Add → Add: Basic → OK
Name it Vocab Builder → OK
Step 2 — Set up fields
Select Vocab Builder → Fields
Anki starts you with a "Front" and "Back" field. Don't delete them — rename them instead:

1. Rename Front → Word
2. Rename Back → Definition
3. Click Add → name it Example
4. Click Add again → name it Audio

Final order: Word · Definition · Example · Audio
Step 3 — Card Template 1: Word → Definition
Select Vocab Builder → Cards → rename to Word → Definition
Paste these into the front and back fields:
Front Template
{{Word}} {{#Audio}}{{Audio}}{{/Audio}}
Back Template
{{FrontSide}}
{{Definition}}
{{Example}}
Step 4 — Card Template 2: Definition → Word
Still in the Cards editor → Options → Add Card Type → name it Definition → Word
Front Template
{{Definition}}
Back Template
{{FrontSide}}
{{Word}} {{#Audio}}{{Audio}}{{/Audio}}
{{Example}}
Step 5 — Import cards into Anki
This is a one-time setup — come back here each time you want to add new cards.

1. In the vocab builder app, go to the Cards tab → Export New Cards
2. In Anki: File → Import → select the exported .txt file
3. Set Note Type to Vocab Builder
4. Separator: Tab
5. Map fields: Field 1 → Word, Field 2 → Definition, Field 3 → Example, Field 4 → Audio (will be empty — filled in the next step)
6. Check Update existing notes where first field matches — prevents duplicates
7. Click Import
Step 6 — Generate audio with AwesomeTTS
Do this after each import to fill the Audio field on your new cards:

1. Open the Anki card browser → select your newly imported cards
2. AwesomeTTS → Add Audio to Selected
3. Source field: Word → Destination field: Audio
4. Choose a TTS service (Google TTS works well) → Generate

This fills the Audio field with [sound:word.mp3] tags, which both card templates play automatically.
